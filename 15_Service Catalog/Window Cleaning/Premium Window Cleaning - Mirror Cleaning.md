# Mirror Cleaning

> [!summary] Quick Summary
> Interior mirror cleaning using the same tools and techniques we use for professional window cleaning — for a clear, streak-free result.
>
> Includes:
> - Removes dust, fingerprints, and smudges
> - Edges and corners wiped down clean
>
> Benefits:
> - Streak-free clarity using the same method we use for windows
> - Safe for all types of mirrors, including large vanities and wall mirrors
> - Leaves your mirrors looking clean, clear, and like they should
>
> Note:
> This is for indoor, accessible mirrors only. Extra-large or hard-to-reach mirrors may be priced differently depending on access.

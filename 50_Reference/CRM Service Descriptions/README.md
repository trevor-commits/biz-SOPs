# CRM Service Descriptions

Drop HCP-exported service description files into this folder for intake.

Workflow:
- Place the raw export here without renaming it unless the filename is unclear.
- Keep the raw export as the source-of-truth intake file.
- Create any import-ready or audit-ready derived files next to the raw export instead of overwriting it.
- After import, convert the source into Obsidian-friendly Markdown notes.
- Keep the original export in this folder as the audit source of truth.
